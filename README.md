# 📊 KPI Platform

Веб-платформа для управления, отслеживания и визуализации KPI  
(ключевых показателей эффективности).

Проект построен на современном frontend-стеке и использует **Firebase**
для хранения данных и управления доступом.

---

## 🚀 Возможности

- 📈 Отслеживание KPI в удобном интерфейсе  
- 📊 Визуализация показателей и динамики  
- 👥 Работа с пользователями и ролями  
- ☁️ Хранение данных в Firebase (Firestore / Storage)  
- 🔐 Правила доступа и безопасность данных  

> Проект находится в активной разработке, функциональность может расширяться.

---

## 🛠️ Технологии

- **JavaScript / TypeScript**
- **Vite**
- **Frontend framework** (React / Vue — в зависимости от реализации)
- **Firebase**
  - Firestore
  - Storage
  - Security Rules
